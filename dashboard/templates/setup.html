Schritt 1 — Claude AI verbinden
Der Bot nutzt Claude AI um Social-Media-Posts zu schreiben. Du brauchst dafür einen API Key von Anthropic — das ist die KI dahinter.
Hole deinen Key hier: console.anthropic.com → API Keys → Create Key
Der Key beginnt immer mit sk-ant-
Schritt 2 — Plattformen wählen
Auf welchen Plattformen soll der Bot posten? Du kannst auch später Plattformen hinzufügen oder entfernen.
{% set plat_defs = [('instagram','📸','Instagram'),('facebook','👍','Facebook'),('linkedin','💼','LinkedIn'),('twitter','🐦','Twitter / X'),('tiktok','🎵','TikTok')] %} {% for pid, icon, name in plat_defs %}
{{ icon }}
{{ name }}
{% endfor %}
Schritt 3 — Plattform-Zugänge
Trage die API-Zugangsdaten für die gewählten Plattformen ein. Klicke auf den Anleitungs-Link für Schritt-für-Schritt-Hilfe.
Schritt 4 — E-Mail Benachrichtigungen
Erhalte eine E-Mail wenn Posts fehlschlagen oder Kunden eskalieren. Dieser Schritt ist optional.
Gmail → Konto → Sicherheit → App-Passwörter → neues Passwort für "Mail" erstellen
